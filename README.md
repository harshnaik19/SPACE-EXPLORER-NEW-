🚀 Space Explorer

Space Explorer is an interactive web application designed to help users explore fundamental concepts of space science and orbital mechanics in a simple and visual way.

The project allows users to calculate orbital velocity and orbital period for different celestial bodies based on altitude. It also provides visual orbit simulations, space facts, and educational content to make learning about space engaging and intuitive.

Key features include:

• 🌍 Orbit Explorer – Calculate orbital velocity and period for planets like Earth and visualize satellite orbits.
• 🛰 ISS Tracker – Displays the position of the International Space Station in real time.
• 📚 Learn Section – Educational explanations about space, gravity, and orbital mechanics.
• ❓ Interactive Quiz – Test knowledge about planets, satellites, and space science.
• 🌌 Space Facts – Random interesting facts about the universe and planetary science.
• 🚀 Rocket Evolution – Visual timeline showing the development of rockets from early space programs to modern launch vehicles.

This project is built using HTML, CSS, and JavaScript with a focus on interactive visualization and educational exploration. It is hosted using GitHub Pages so it can be accessed from anywhere without running a backend server.

Space Explorer is intended as a beginner-friendly educational tool for students, space enthusiasts, and anyone curious about how satellites orbit planets and how space missions work.
